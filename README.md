# Maki 2

Mapbox's new icon set. An eventual replacement for [https://github.com/mapbox/maki](maki).

### Format

- Icons are available in two sizes: 11 x 11px and 15px x 15px.
- Icons are each a single unified SVG path, with no transforms, groups or strokes, and a #000000 fill.

### Design guidelines

- Use the simplest, most universally recognizable symbol for each icon.
- All icons should share a consistent level of detail and visual weight.
- Design with the pixel grid in mind so icons look sharp at 1x resolution.
- Make sure icons are visually centered inside the canvas so they look good when placed behind a shield or marker.
- Unless the design calls for a sharp edge, corners should be consistently rounded for a friendlier appearance.

## Work Flow

All maki 1 icons have gone through an initial design step of being re-drawn for maki 2. Next steps will be to refine these already drawn maki 2 icons, as well as draw new icons from the following table. As requests for new icons emerge, they will be added to this list.
